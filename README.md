# 🔥 SnackBurn — 군것질, 똑똑하게 태우자

> 식사 외에 먹은 군것질을 기록하세요. SnackBurn이 칼로리를 계산하고, 그만큼을 태우려면 어떤 운동을 얼마나 해야 하는지 알려줍니다. 죄책감 없이, 그저 알아차림으로.

## 라이브 데모

🔗 **배포 URL:** *https://ai-as-4.vercel.app/*

---

## 주요 기능

- **군것질 기록** — 군것질 이름과 개수를 입력해 추가
- **칼로리 자동 계산** — 자주 먹는 군것질 100여 종의 내장 데이터베이스로 즉시 계산
- **직접 입력 fallback** — 목록에 없는 군것질은 1개당 칼로리를 직접 입력
- **실시간 누적 합계** — 오늘의 군것질 칼로리가 즉시 합산되고, 권장량 대비 진행 바 표시
- **운동 변환** — 하루 총합을 4가지 운동 플랜(걷기/조깅/자전거/줄넘기)으로 변환 (MET 공식 기반)
- **날짜 자동 초기화** — 날짜가 바뀌면 기록이 자동으로 초기화됨
- **삭제 / 전체 삭제** — 군것질 하나만 삭제하거나, 하루 전체를 초기화(확인 절차 포함)
- **백엔드 불필요** — 모든 데이터는 브라우저 `localStorage`에 저장
- **모바일 친화적** — 완전 반응형 레이아웃

| 페이지 | 파일 | 설명 |
|------|------|-------------|

| 운동 | `workout.html` | 오늘의 칼로리를 태우는 운동 플랜 |

## 로컬에서 실행하기

1. **레포지토리 클론**
   ```bash
   git clone https://github.com/YOUR_USERNAME/snackburn.git
   cd snackburn
   ```

2. **브라우저에서 열기**
   ```bash
   # 방법 A: index.html을 바로 열기
   open index.html

   # 방법 B: 로컬 서버 사용 (권장)
   npx serve .
   # 또는
   python3 -m http.server 8080
   ```

3. **빌드 단계 불필요** — 순수 HTML / CSS / JavaScript.

## 기술 스택

- **HTML5 / CSS3 / 바닐라 JavaScript** — 프레임워크·npm 없음
- **내장 칼로리 데이터베이스** — `snackdata.js`에 자주 먹는 군것질 100여 종 수록 (키워드 매칭)
- **MET 운동 공식** — 표준 MET 값으로 칼로리당 운동 시간 계산
- **localStorage** — 날짜 자동 초기화 기능이 있는 클라이언트 저장
- **Vercel** — 설정 없는 정적 사이트 배포 (외부 API·서버 불필요)

> 칼로리 데이터베이스와 운동 공식은 AI(Claude)의 도움을 받아 구성했습니다.

## 배포 (Vercel)

1. 이 레포지토리를 GitHub에 푸시합니다.
2. [vercel.com](https://vercel.com) → **Add New Project** 클릭.
3. GitHub 레포를 import 합니다.
4. 모든 설정을 기본값으로 둡니다 (Vercel이 정적 사이트를 자동 인식).
5. **Deploy** 클릭 — 약 30초 후 사이트가 공개됩니다.

> 외부 API 키나 환경변수 설정이 전혀 필요 없습니다. 배포 즉시 작동합니다.
